## 一、项目概述

### 1.1 优化目标

为宠物健康提醒功能添加用户自定义能力，达到美国主流宠物健康 App 标准：
1. **提醒时间自定义** - 用户可设置提醒发送的具体时间
2. **提醒天数自定义** - 用户可选择提前多少天收到通知
3. **提醒声音选择** - 用户可从多种声音中选择

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 单一职责 | 一个功能一个模块，高内聚低耦合 |
| 代码复用 | 优先复用现有模块，抽象符合"三次法则" |
| 最小代码 | 用最少代码实现功能，避免过度设计 |
| 编译验证 | 每次改动后自动编译验证 |
| 清理废弃 | 替换后标记废弃 → 验证无影响 → 删除 |

### 1.3 与当前项目切合点

| 现有模块 | 复用方式 |
|----------|---------|
| `NotificationService` | 扩展新方法，保留向后兼容 |
| `VaccineRecord` | 新增提醒配置字段 |
| `Medication` | 新增声音配置字段 |
| `AddVaccineView` | 新增提醒设置 Section |
| `AddMedicationView` | 新增声音选择和多时间支持 |
| `PetDetailViewModel` | 复用现有数据管理逻辑 |

**注意**：现有 `Medication` 模型使用 `reminderTimesData` 作为存储字段，通过 computed property 转换。
新增字段需遵循相同模式以保持一致性。

**新增字段**：

```swift
// 文件：pethealthdata/pethealthdata/Models/Medication.swift

/// 提醒声音文件名（持久化存储）
var notificationSoundData: Data?

/// 提醒声音文件名（计算属性）
/// 默认值："bamboo.caf"
var notificationSound: String {
    get {
        guard let data = notificationSoundData,
              let sound = try? JSONDecoder().decode(String.self, from: data) else {
            return "bamboo.caf"
        }
        return sound
    }
    set {
        notificationSoundData = try? JSONEncoder().encode(newValue)
    }
}
```

**生成规则**：
1. 遵循现有 `reminderTimes` 的存储模式
2. 添加存储字段 `notificationSoundData`
3. 添加计算属性 `notificationSound`
4. **字段顺序**：放在 `notes` 之后、`createdAt` 之前（与 reminderTimesData 位置一致）

**代码示例**：

/// 提醒声音配置
enum NotificationSoundConfig {
    
    /// 声音选项
    struct SoundOption: Identifiable, Hashable {
        let id: String
        let name: String
        let fileName: String
        let previewIcon: String
        
        static let allCases: [SoundOption] = [
            .init(id: "triTone", name: "Tri-Tone", fileName: "triTone.caf", previewIcon: "music.note"),
            .init(id: "bamboo", name: "Bamboo", fileName: "bamboo.caf", previewIcon: "music.note"),
            .init(id: "default", name: "Default", fileName: "default.caf", previewIcon: "bell"),
            .init(id: "note", name: "Note", fileName: "note.caf", previewIcon: "music.note"),
            .init(id: "pop", name: "Pop", fileName: "pop.caf", previewIcon: "bubble.left"),
            .init(id: "sonar", name: "Sonar", fileName: "Sonar.caf", previewIcon: "waveform"),
            .init(id: "silent", name: "Silent", fileName: "", previewIcon: "bell.slash")
        ]
        
        static let `default` = SoundOption(id: "triTone", name: "Tri-Tone", fileName: "triTone.caf", previewIcon: "music.note")
        static let medicationDefault = SoundOption(id: "bamboo", name: "Bamboo", fileName: "bamboo.caf", previewIcon: "music.note")
    }
    
    /// 获取声音文件 URL
    static func soundURL(for fileName: String) -> URL? {
        guard !fileName.isEmpty else { return nil }
        return Bundle.main.url(forResource: String(fileName.dropLast(4)), withExtension: "caf")
    }
    
    /// 检查声音文件是否存在
    static func soundExists(fileName: String) -> Bool {
        guard !fileName.isEmpty else { return true } // silent
        return soundURL(for: fileName) != nil
    }
}

import SwiftUI

/// 提醒提前天数多选器
struct ReminderDaysPicker: View {
    @Binding var selectedDays: [Int]
    
    private let options = VaccineReminderConfig.ReminderTimeOption.allCases
    
    var body: some View {
        ForEach(options) { option in
            HStack {
                Text(option.displayName)
                Spacer()
                Image(systemName: selectedDays.contains(option.days) ? "checkmark.circle.fill" : "circle")
                    .foregroundColor(.accentColor)
            }
            .contentShape(Rectangle())
            .onTapGesture {
                toggleDay(option.days)
            }
        }
    }
    
    private func toggleDay(_ days: Int) {
        if selectedDays.contains(days) {
            selectedDays.removeAll { $0 == days }
        } else {
            selectedDays.append(days)
            selectedDays.sort()
        }
    }
}
